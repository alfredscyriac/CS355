### Live preview of md file in vscode 
> Requires Markdown All in One extension by Yu Zhang
```bash
Cmd+Shift+V
```

### Convert md to pdf in vscode 
> Requires Markdown PDF extension by yzane 
1. Cmd+Shift+P 
2. Type export 
3. Select 'markdown-pdf: Export (pdf)'
